## 🚀 How to Run Locally

```bash
# 1. Clone the repository
git clone https://github.com/YOUR_USERNAME/100-Days-DataScience-Projects.git

# 2. Navigate into the folder
cd 100-Days-DataScience-Projects

# 3. Install required libraries
pip install -r requirements.txt

# 4. Launch Jupyter Notebook
jupyter notebook
```
